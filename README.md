# Курсовая работа по предмету ООП

Данный проект представляет собой минимальный веб-монитор ресурсов. Основная цель — демонстрация навыков объектно-ориентированного программирования и работы с веб-технологиями.

# Важно:
проект находится на стадии MVP. Код упрощён, содержит многочисленные костыли и специально выложен в экспериментальной форме. Проект предназначен исключительно для личного использования и не ориентирован на публичное применение.

# Требования

Для работы необходимы библиотеки `psutil` и `Flask`. Установка:

#### pip:
```pip
pip install psutil Flask
```

#### Debian*:
```bash
apt install python-psutil python-flask
```

#### RHEL*:
```
dnf install python-psutil python-flask
```

#### Arch*:
```
pacman -S python-psutil python-flask
```

# Запуск

Для запуска веб-сервера выполнить:
```bash
python3 webserver.py
```

По умолчанию интерфейс доступен по адресу http://localhost:5001. Интерфейс демонстрирует базовые метрики системы: загрузку процессора, использование памяти и простые графики.

# Ограничения
1. Проект реализован как минимальный прототип, без строгого разделения на модули.
2. Функционал ограничен базовыми метриками и простыми визуализациями.
3. Код содержит многочисленные упрощения и костыли.
4. Проект предназначен исключительно для личного использования и изучения.
